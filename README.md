# סורק מניות

אתר סטטי שמציג גרף מחיר, ממוצעים נעים, נפח מסחר ו-RSI עבור מניה שמוקלדת.

## תכונות

- גרף נרות יפניים עם 3 ממוצעים נעים ניתנים להגדרה (ברירת מחדל 20 / 150 / 200)
- חיווי חץ ▲/▼ ליד כל ממוצע: האם המחיר מעל או מתחת לממוצע הנע
- גרף נפח מסחר (Volume) מסונכרן עם גרף המחיר
- גרף RSI(14) עם קווי 30/70
- בחירת טווח תצוגה: חודש, חודשיים, 3 חודשים, חצי שנה, שנה, שנתיים, 5 שנים, מקסימום
- תמיכה בסימולים אמריקאיים (AAPL) וישראליים (TEVA.TA)

## הרצה לוקלית

הקובץ סטטי לחלוטין. אפשרויות הפעלה:

```bash
# Python
python -m http.server 8000

# Node
npx serve .
```

ופותחים `http://localhost:8000/`.

## פריסה ל-GitHub Pages

1. דחיפת תוכן התיקייה לריפו ב-GitHub.
2. Settings → Pages → Source: Deploy from a branch → main / root → Save.
3. ממתינים כדקה ומקבלים URL ציבורי.

## מקור נתונים

הנתונים מגיעים מ-Yahoo Finance Chart API דרך CORS proxies ציבוריים
(corsproxy.io, allorigins.win, codetabs). אין צורך במפתח API.
